VSCrib.vim
================================================================================
A VimL library for cribbing project infrastructure and workspace configurations
from VSCode.

Installation
--------------------------------------------------------------------------------

With vim-plug,

```vim
call plug#begin('~/.vim/bundle')
" ...
Plug 'Yilin-Yang/VSCrib.vim'

  " dependencies
  Plug 'Google/vim-maktaba'

" ...
call plug#end()
```
